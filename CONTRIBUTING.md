# Contributing to Positronic

Thank you for your interest in contributing to the Positronic blockchain!

## How to Contribute

### Reporting Bugs

- Use the [Bug Bounty Program](https://positronic-ai.network/bug-bounty/) for security vulnerabilities
- Open an issue in the [Community Repository](https://github.com/PositronicAI-Blockchain-L1/community/issues) for general bugs

### Feature Requests

- Join our [Discord](https://discord.gg/DRyKBkbahu) and discuss in `#dev-chat`
- Open a discussion in the [Community Repository](https://github.com/PositronicAI-Blockchain-L1/community/discussions)

### Running a Node

The easiest way to contribute is by running a node:

1. Download the desktop application from [positronic-ai.network/download](https://positronic-ai.network/download/)
2. Get testnet ASF from the [faucet](https://positronic-ai.network/faucet/)
3. Stake 32 ASF to become a validator

### Development

For SDK development and integrations, see:

- [JavaScript SDK Documentation](https://positronic-ai.network/docs/)
- [Python SDK Documentation](https://positronic-ai.network/docs/)
- [Telegram SDK Documentation](https://positronic-ai.network/telegram/)

## Code of Conduct

Be respectful, constructive, and professional in all interactions.
